# Andrew Ng Machine Learning Specialization — From Scratch

This repository contains my implementations, experiments, and notes while completing the Machine Learning Specialization by Andrew Ng.

Rather than only completing the course notebooks, I reimplement core machine learning algorithms from scratch in Python to build a deeper understanding of the underlying mathematics and optimization techniques.

## Objectives

* Understand machine learning fundamentals mathematically
* Implement algorithms from scratch using NumPy
* Reproduce key results from course labs
* Build a reusable ML codebase
* Develop research-oriented coding practices

## Running the Project

Clone the repository:

```bash
git clone https://github.com/ujjwal77771/andrew-ng-ml.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

or run Python scripts directly:

```bash
python main.py
```
